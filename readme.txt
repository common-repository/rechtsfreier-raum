== Description ==

Dieses Plugin zeigt einen kleinen rechtsfreien Raum auf deinem Blog an. Das Plugin muss nicht konfiguriert werden.

== Installation ==

1. Lade den Ordner "rechtsfreier-raum" mit seinem Inhalt in den Ordner "/wp-content/plugins/".

2. Aktiviere das Plugin ueber das Plugin-Menue im Wordpress Adminpanel.
